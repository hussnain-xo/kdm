Kalupura Download Manager — release builds
========================================

1) Install build deps (use a venv on each OS you build on):
   pip install -r requirements-build.txt

2) Build on each target OS (PyInstaller does not cross-compile):
   Windows (folder + zip):     scripts\build_windows.bat
   Windows (installer EXE):    scripts\build_windows_installer.ps1
                               Requires Inno Setup 6 (Programs & Features uninstall).
   macOS:    chmod +x scripts/build_macos.sh && ./scripts/build_macos.sh
   Linux:    chmod +x scripts/build_linux.sh && ./scripts/build_linux.sh

   One-shot release (macOS/Linux):  ./scripts/package_release.sh

3) End-user quick guide (short steps for Mac / Windows / Linux):
   packaging/USER_QUICK_START.txt — release scripts copy it into dist/ and DMG/ZIP as
   USER_QUICK_START.txt. First app launch shows a one-time “Browser integration” prompt.

4) Outputs:
   Windows: dist\KDM\KDM.exe — portable folder; dist\release\KDM-Setup-<ver>-x64.exe — installer
              (installs under Program Files, includes browser-extension next to KDM.exe).
   macOS:   dist/release/KDM-<ver>-macOS.dmg — drag KDM.app to Applications
            dist/release/KDM-Setup-<ver>-macOS.pkg — installs into /Applications (optional)
   Linux:   dist/release/KDM-<ver>-Linux-x86_64.tar.gz — extract and run ./KDM/KDM
            dist/release/kdm_<ver>_amd64.deb — sudo apt install ./kdm_*.deb (optional)

5) Browser extension (Chrome / Edge — same as manual “Load unpacked”):
   The PyInstaller build copies extension-for-users/KDM-Browser-Extension into
   browser-extension/ next to the app. After install, run:
     KDM.exe --install-browser-extension   (Windows)
     ./KDM --install-browser-extension     (Linux)
     Open KDM.app from Terminal: .../KDM.app/Contents/MacOS/KDM --install-browser-extension
   Or use the installer’s post-install step (Windows). Chrome/Edge cannot silently install
   unpacked extensions without enterprise policy; users enable Developer mode once.

6) Playwright / Chromium:
   Sites that need browser extraction require Chromium once per machine:
   playwright install chromium

7) Selling online:
   Put your download links in kdm_config.json as "distribution_page".
   The in-app Share button copies that link and a short promo message.

8) Code signing (optional, recommended for macOS/Windows):
   macOS: codesign / notarize the .app before making the DMG
   Windows: sign KDM.exe with your Authenticode certificate
