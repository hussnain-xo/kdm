================================================================================
  Kalupura Download Manager (KDM) — Browser extension (Chrome / Edge / Firefox)
================================================================================

BEFORE YOU START
  • Install and OPEN the KDM desktop app on your computer first.
  • The extension talks to KDM at http://127.0.0.1:9669 — the app must be running.

--------------------------------------------------------------------------------
CHROME (Windows / Mac / Linux)
--------------------------------------------------------------------------------
  1. Unzip this package if you received a ZIP file.
  2. Open Google Chrome.
  3. In the address bar type:  chrome://extensions
  4. Turn ON “Developer mode” (top right).
  5. Click “Load unpacked”.
  6. Select the folder named “KDM-Browser-Extension” (the folder that contains
     manifest.json — NOT a file inside it).

--------------------------------------------------------------------------------
MICROSOFT EDGE
--------------------------------------------------------------------------------
  1. Unzip if needed.
  2. Open Edge and go to:  edge://extensions
  3. Turn ON “Developer mode” (left side).
  4. Click “Load unpacked” and choose the “KDM-Browser-Extension” folder.

--------------------------------------------------------------------------------
MOZILLA FIREFOX (Manifest V3 — same folder as Chrome)
--------------------------------------------------------------------------------
  1. Open Firefox 128+ (same extension files: manifest.json + scripts in this folder).
  2. Address bar:  about:debugging#/runtime/this-firefox
  3. Click “Load Temporary Add-on…” and select manifest.json inside KDM-Browser-Extension.
  Note: Temporary add-ons clear when Firefox fully exits — use again next session,
  or Mozilla enterprise policies for permanent install.

--------------------------------------------------------------------------------
TROUBLESHOOTING
--------------------------------------------------------------------------------
  • “KDM server is not running” → Start the KDM application, then refresh the page.
  • Extension missing after browser update → Load unpacked again with the same folder.
  • Safari: this ZIP is for Chrome/Edge; Safari needs a separate setup (see KDM docs).
